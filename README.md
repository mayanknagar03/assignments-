## About

This repository contains solutions to the End-Semester Practical Examination assignments for the Data Structures & Algorithms lab. Each assignment was implemented in C and covers core DSA concepts including sorting algorithms, linked lists, stacks, queues, binary search trees, and heaps.

---

## Assignments

### 1. Sorting, Array Reversal & Doubly Linked List
**File:** `assignment-1-doubly-linked-list/doubly_linked_list_sorting.c`

- Applied **Heap Sort** to sort a digit array in non-decreasing order
- Performed **in-place array reversal**
- Constructed a **Doubly Linked List** from the reversed array
- Traversed the list in both **forward and backward** directions

---

### 2. Sorting, Stack & Queue Operations
**File:** `assignment-2-stack-queue/stack_queue_sorting.c`

- Applied **Bubble Sort** on a digit array
- Implemented a **Stack** (push/pop) using arrays
- Implemented a **Queue** (enqueue/dequeue) using arrays
- Demonstrated data transfer from Stack to Queue maintaining reversal order

---

### 3. Binary Search Tree — Traversals, Level Order & Node Metrics
**File:** `assignment-3-bst-traversals/bst_traversals.c`

- Built a **Binary Search Tree (BST)** from digit input
- Printed **Inorder** and **Preorder** traversals
- Performed **Level Order Traversal** using an array-based Queue
- Calculated and printed **height and depth** of each node using Post Order traversal

---

### 4. BST with Min/Max Heap Construction & Deletion
**File:** `assignment-4-bst-heap/bst_heap_operations.c`

- Constructed a **BST** and printed Inorder and Post Order traversals
- Built a **Min Heap or Max Heap** conditionally based on roll number parity
- Implemented **root deletion** followed by heapification
- Printed the updated heap array after deletion
